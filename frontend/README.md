# Frontend Portfolio App

This is the React frontend for HammadDevx's portfolio website.

## Tech

- React 19
- CRACO
- Tailwind CSS
- Framer Motion
- Lucide React

## Available Scripts

### `npm start`

Runs the app in development mode at `http://localhost:3000`.

### `npm run build`

Builds the app for production into the `build/` directory.

### `npm test`

Runs the test command through CRACO.

## Important Files

- `public/index.html`  
  Main HTML template and favicon registration

- `public/favicon.svg`  
  Custom SVG favicon for the portfolio

- `src/mock.js`  
  Central content source for profile info, services, projects, testimonials, and tech stack

- `src/components/`  
  Portfolio sections and shared UI components

## Design Notes

- Supports light and dark mode
- Responsive across mobile, tablet, and desktop layouts
- Uses motion and glass/surface card patterns throughout the interface
